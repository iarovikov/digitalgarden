# Как переехать с ChatGPT на Claude

Claude запоминает тебя между разговорами, хранит историю чатов и даже умеет импортировать память из ChatGPT. Переехать можно за 5 минут.

---

## Способ 1: Быстрый импорт памяти (5 минут)

Это самый простой способ — Claude заберёт из ChatGPT твои предпочтения, стиль общения и контекст.

1. Открой Claude → **Settings** (значок шестерёнки) → **Capabilities** → **Memory**
2. Нажми **Start Import**
3. Claude покажет специальный текст — скопируй его
4. Открой ChatGPT в новом окне и создай новый чат
5. Вставь скопированный текст в ChatGPT и отправь
6. ChatGPT сгенерирует сводку о тебе — скопируй её
7. Вернись в Claude и вставь эту сводку в окно импорта
8. Готово! Claude запомнит твои предпочтения

Также можно открыть напрямую: **claude.ai/import-memory**

---

## Способ 2: Перенос через экспорт данных (если нужна полная история)

1. Открой **ChatGPT** → нажми на аватар (внизу слева) → **Settings**
2. Перейди в **Data controls** → **Export data** → **Confirm export**
3. На почту придёт письмо со ссылкой — скачай ZIP-архив
4. Распакуй архив — внутри будет файл **chat.html**
5. В Claude создай **Project** (проект) и загрузи туда chat.html

> Ограничение: файл не больше 30 МБ. Если больше — загрузи только нужные части.

---

## Способ 3: Перенос отдельных чатов вручную

Если нужен только один конкретный разговор:

1. Открой нужный чат в ChatGPT
2. Выдели текст → скопируй (Ctrl+C)
3. В Claude создай новый чат → вставь (Ctrl+V)
4. Напиши: *«Вот наш предыдущий разговор, продолжим»*

Или сохрани текст в файл (.txt) и прикрепи его через скрепку в Claude.

## Что умеет Claude (чего ты мог не знать)

| Функция | Как работает |
|---------|-------------|
| **Память** | Claude запоминает тебя между чатами — имя, стиль, предпочтения. Обновляется каждые 24 часа |
| **История чатов** | Все разговоры сохраняются в боковой панели, можно искать |
| **Поиск по чатам** | Можно попросить Claude найти что-то из прошлых разговоров |
| **Проекты** | Отдельные рабочие пространства с файлами и инструкциями. Все чаты внутри проекта видят загруженные файлы |
| **Управление памятью** | Settings → Memory — можно посмотреть и отредактировать, что Claude о тебе помнит |

## Советы для начала

- **Настрой профиль** — Settings → Profile. Напиши о себе: чем занимаешься, как хочешь общаться. Claude будет учитывать это во всех чатах
- **Создай проекты** — для каждой темы (работа, учёба, хобби) создай отдельный проект и загрузи нужные файлы
- **Не переноси всё** — 90% старых чатов уже не нужны. Импорт памяти (Способ 1) перенесёт самое важное
- **Язык** — Claude отлично понимает русский, можно писать как удобно
